## 🚀 طريقة الاستخدام والتطبيق (How to Use)

### 📂 الخطوة 1: استخراج اللانشر
1. قم بتحميل ملف الـ Zip الخاص باللانشر من الجدول أعلاه.
2. فك الضغط عن مجلد `PrismLauncher-Windows-MSVC-Portable-9.4`.

### 🛠️ الخطوة 2: تشغيل السكربت وتوليد الحساب
1. ضع ملف `QOQ.bat` داخل المجلد المستخرج (بجانب ملف `PrismLauncher.exe`).
2. اضغط مرتين لتشغيل ملف `QOQ.bat`.
3. أدخل **اسم المستخدم (Username)** الذي ترغب باللعب به، ثم اضغط `Enter`.
4. سيقوم السكربت تلقائياً بإنشاء ملف جديد باسم `accounts.json` في نفس المجلد.

### 🎲 الخطوة 3: اللعب
1. افتح لانشر `PrismLauncher.exe`.
2. ستجد الحساب الأوفلاين مضافاً وجاهزاً للاستخدام مباشرة!
